h2. Description

p(. The calls that are not related to any user are not stored in this table. One record from the [[DB_DIVOS_STATISTIK_2026_0_CallsQueues|CallsQueues]] table corresponds to one record in this table. The call is considered as related to a user if [[DB_DIVOS_STATISTIK_2026_0_Calls|Calls.AgentNumber]] is set, or [[DB_DIVOS_STATISTIK_2026_0_CallsQueues|CallsQueues.AgentNumber]] is set, or [[DB_DIVOS_STATISTIK_2026_0_CallsQueues|CallsQueues.QueueNumber]] is a service that is directly assigned to the user. If the call is fixed forwarded from the service that is directly assigned to the user, it is not written. This table is used exclusively by the *History* plugin of *Communication Client*.
